# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records as described in [ADRs](https://adr.github.io/).

## Index

- [ADR-0001 – SQLite as master database](./ADR-0001.md)
- [ADR-0003 – Reusable business logic](./ADR-0003.md)
- [ADR-0004 – Audit fields on persistent entities](./ADR-0004.md)

## Process

1. Every architectural decision must be documented as an ADR
2. ADRs are immutable once accepted
3. Implementation must never contradict an accepted ADR
4. If implementation requires architectural change, propose a new ADR for approval
