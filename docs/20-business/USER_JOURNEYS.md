# User Journeys

## Introduction

User Journeys describe how AlpenTind creates value in operational work.

Engineering Specifications (ESR) implement parts of a User Journey.

User Journeys are business documentation, not implementation specifications.

---

## Eight User Journeys

### UJ-001 — First Contact

**Purpose:** Understand the person

**Outcome:** An active dialogue

**Implemented by:**
- ESR-001 Arbetsdag
- ESR-002 Dialog

---

### UJ-002 — Professional Assessment

**Purpose:** Develop sufficient understanding to make a professional assessment

**Outcome:** Professional assessment completed

**Future ESR:** ESR-003 Professional Assessment

---

### UJ-003 — Professional Recommendation

**Purpose:** Present AlpenTind's professional recommendation

**Outcome:** Shared understanding

**Future ESR:** ESR-004 Professional Recommendation

---

### UJ-004 — Shared Decision

**Purpose:** Reach agreement

**Outcome:** Mutual commitment

**Future ESR:** ESR-005 Shared Decision

---

### UJ-005 — Booking

**Purpose:** Register the agreed resa

**Outcome:** Operational booking

**Future ESR:** ESR-006 Booking

---

### UJ-006 — Planning

**Purpose:** Prepare the resa

**Outcome:** Everyone is ready

**Future ESR:** ESR-007 Planning

---

### UJ-007 — Resa

**Purpose:** Deliver the resa

**Outcome:** Professional execution

**Future ESR:** ESR-008 Resa

---

### UJ-008 — Reflection

**Purpose:** Capture learning

**Outcome:** Improved professional judgement

**Future ESR:** ESR-009 Reflection

---

## Journey Map

```text
UJ-001 First Contact
  -> UJ-002 Professional Assessment
  -> UJ-003 Professional Recommendation
  -> UJ-004 Shared Decision
  -> UJ-005 Booking
  -> UJ-006 Planning
  -> UJ-007 Resa
  -> UJ-008 Reflection
```

---

## ESR Roadmap Connection

The User Journeys define AlpenTind's operational value flow from first contact to reflection.

The ESR roadmap implements this flow incrementally:

- UJ-001 is currently implemented by ESR-001 Arbetsdag and ESR-002 Dialog.
- UJ-002 maps to future ESR-003 Professional Assessment.
- UJ-003 maps to future ESR-004 Professional Recommendation.
- UJ-004 maps to future ESR-005 Shared Decision.
- UJ-005 maps to future ESR-006 Booking.
- UJ-006 maps to future ESR-007 Planning.
- UJ-007 maps to future ESR-008 Resa.
- UJ-008 maps to future ESR-009 Reflection.
